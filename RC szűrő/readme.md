# RC szűrő kapcsolás(Alul áteresztő)    

## Születési idő: 2006.08.20.    
---
### Adatok
$R= 8ohm$    
$C= 20uF$      
---  
  
$$fh = {1 \over (2 * π * R * C)}$$  
$$fh = {1 \over (2 * π * 20 uF * 8 ohm)}=994,718$$
